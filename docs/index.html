スクロール

01 / コンセプト

光と機構の
交差点

5節平面リンク機構のクーラー点にWS2812B LEDを搭載。2つのマイクロサーボが角度を変えることで、逆運動学がLEDをグリフの筆跡に沿って正確に誘導します。

表示機（Display）はESP-NOWで受信したカメラフレームを画素ごとの最大輝度合成で積算し、長時間露光による発光書道の映像を生成します。

L1 = L2 = L3 = L4 = 60 mm  ·  D = 36 mm
ワークスペース中心 ≈ (18, 75) mm

02 / システム

2台のデバイス、
1本の無線リンク

Drawing（描画機）

M5Stack AtomS3R-CAM

  • MCUESP32-S3
  • サーボマイクロサーボ × 2 — G25, G26
  • LEDWS2812B × 1 — G0
  • リンクL1=L2=L3=L4=60mm, D=36mm
  • 役割逆運動学計算・サーボ駆動・LED制御・カメラ送信

Display（表示機）

M5Stack AtomS3R

  • MCUESP32-S3
  • 入力ジョイスティック (G8/G7) + ボタン3個
  • 画面内蔵 128×128 LCD
  • 通信ESP-NOW（無線）
  • 役割長時間露光合成・メニューUI・REC/PLAY
ESP-NOW カメラフレーム + グリフパケット

04 / ハードウェア

部品リスト

描画機（Drawing）

M5Stack AtomS3R-CAM ESP32-S3 · メインコントローラ + OV2640カメラ
マイクロサーボ × 2 G25（サーボ1）/ G26（サーボ2）
WS2812B LED × 1 G0 · リンク機構クーラー点に搭載
4サーボ接続基板 ［自作］ サーボ電源供給ボード → スイッチサイエンス ssci.to/11122
5節リンク機構フレーム 自作 · 全リンク長 60mm、ピボット間隔 D = 36mm

表示機（Display）

M5Stack AtomS3R ESP32-S3 · コントローラ + 内蔵 128×128 LCD
ミニコントローラ ［自作］ ジョイスティック (G8/G7) + ボタン3個、ハンドヘルド型 → スイッチサイエンス ssci.to/9520

pio run -e Display --target upload
